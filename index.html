Pertanyaan yang Sering Diajukan

1. Apa itu Code Craft Studio?

Code Craft Studio adalah platform digital yang menyediakan template website portofolio, landing page, dan proyek siap pakai untuk para freelancer, pelajar, maupun pebisnis digital.

2. Bagaimana cara membeli template?

Klik tombol "Beli Sekarang" pada template yang kamu inginkan. Kamu akan diarahkan ke WhatsApp kami untuk menyelesaikan proses pembayaran dan pengiriman file.

3. Apa metode pembayaran yang didukung?

Kami menerima pembayaran melalui QRIS All Payment dari aplikasi DevPay. Praktis, cepat, dan aman.

4. Setelah beli, file dikirim lewat mana?

Setelah pembayaran dikonfirmasi, file template akan dikirim melalui Google Drive atau link unduhan khusus yang bersifat private.

5. Bolehkah saya menggunakan template ini untuk client saya?

Ya, kamu boleh menggunakan template untuk proyek client yang kamu kerjakan secara pribadi. Namun, template ini tidak boleh diperjualbelikan ulang secara mentah, misalnya diupload ulang ke marketplace, diklaim sebagai karya sendiri, atau dijual dalam bentuk bundling produk digital.

6. Apakah saya bisa request template custom?

Tentu saja! Hubungi kami via WhatsApp dan tim kami akan bantu pembuatan template custom sesuai kebutuhan kamu.
